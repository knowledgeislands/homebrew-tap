# knowledgeislands/homebrew-tap

Homebrew formulae for [Knowledge Islands](https://github.com/knowledgeislands) command-line tools.

## Usage

```sh
brew install knowledgeislands/tap/<formula>
```

Homebrew reads `knowledgeislands/tap` as this repository (`homebrew-tap`), so no explicit `brew tap` is needed.

## Formulae

| Formula | Description | Source |
| --- | --- | --- |
| `mgit` | Run commands across many git repositories at once. | [tools-mgit](https://github.com/knowledgeislands/tools-mgit) |
